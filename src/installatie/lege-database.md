# Starten met lege database

In de map met het installatiebestand maakt Abacus het databasebestand *db.sqlite* aan waarin alle verkiezings- en gebruikersgegevens worden opgeslagen.
Gooi dit bestand weg als je deze gegevens wilt verwijderen en Abacus wilt starten met een lege database.

Je kunt ook de database resetten door Abacus te starten met het argument `-r` of `--reset-database`.
