# Stembureau invoeren

Het invoeren van een stembureau betekent dat je het papieren proces-verbaal precies overneemt in Abacus, zodat er een exacte digitale kopie van de resultaten is. In de volgende stappen lees je hoe je een stembureau invoert. Gebruik hierbij de [cheatsheet](../../cheatsheet/) met daarin de volgende informatie:

- [Hoe je kunt invoeren en welke sneltoetsen je gebruikt](../../cheatsheet/invoeren-sneltoetsen.md)
- [Wat je moet doen bij waarschuwingen en fouten](../../cheatsheet/waarschuwingen-fouten.md)
- [Hoe je een tweede invoer afrondt](../../cheatsheet/tweede-invoer.md)
- [Hoe je de invoer kunt pauzeren of afbreken](../../cheatsheet/pauzeren-afbreken.md)

 **Let op:** je mag het papieren proces-verbaal nooit aanpassen, ook niet als je fouten ziet. Neem de invoer precies over zoals het op het papier staat.
