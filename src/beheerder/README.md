# Instructies voor de beheerder

Welkom bij Abacus! Deze tool is ontwikkeld om de verkiezingsuitslagen per stembureau in te voeren en te controleren op fouten en afwijkende uitslagen.
Na de installatie van Abacus voeg je een verkiezing toe en importeer je stembureaus.
Ook maak je gebruikers aan en kun je ze wijzigen als dit nodig is.
In dit document leggen we uit hoe je Abacus instelt en gebruikt als beheerder.

**Let op:**

- Zet adblockers uit of voeg Abacus toe aan de lijst met uitzonderingen in de adblocker. Kijk hiervoor bij [Gebruik geen adblocker](/installatie/browser.md#gebruik-geen-adblocker).
- Zorg ervoor dat je geen wachtwoordmanager gebruikt. Kijk hiervoor bij [Gebruik geen wachtwoordmanager](/installatie/browser.md#gebruik-geen-wachtwoordmanager).
